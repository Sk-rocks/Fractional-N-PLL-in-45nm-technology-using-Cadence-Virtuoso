# Fractional-N-PLL-in-45nm-technology-using-Cadence-Virtuoso

This project presents the design and analysis of a CMOS Fractional-N Phase-Locked Loop (PLL) designed for wideband wireless applications. Implemented in a 45nm technology using Cadence Virtuoso, the design focuses on core sub-blocks including a zero dead-zone Phase Frequency Detector (PFD), a wide-swing cascode Charge Pump (CP), a ring-based Voltage-Controlled Oscillator (VCO), a Multi-Modulus Frequency Divider (MMFD), and a third-order MASH 1-1-1 Sigma-Delta Modulator. The architecture aims to achieve fine frequency resolution, low phase noise, and a wide tuning range to meet the demands of modern communication standards like 5G and Wi-Fi. Individual block functionality was successfully verified through simulation, establishing a foundation for a fully integrated, high-performance frequency synthesizer capable of supporting advanced wireless systems.
